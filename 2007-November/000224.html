stucked, if we hold on those old things.

Modern PCs could definitely handle a new lincity-ng (and just having a game 
that isn't playable with the old gui anymore doesn't mean the old version 
can't be played at all - you'd have to choose then, but that's ok imho [We 
could create a converter for i.e. old savegame → new savegame, then it's only 
backwards compatibility which is broken).
