There's one other minor quality issue, each of these classes should derive (privately) from an Uncopyable class (like boost::noncopyable) to prevent generation of default copy constructors and the default assignment operator (also in Effective C++ in Item 6).  This is a nice way to prevent accidents :)

class Uncopyable: {
private:
Uncopyable(const Uncopyable&);
Uncopyable &operator=(const Uncopyable&);
};
