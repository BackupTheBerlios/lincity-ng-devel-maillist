I made a more complete test today and I found some problems with the Brazilian Translation.
Some texts were bigger than the tabs and other ugly things in the main screen. I fixed these problems in this patch.

Another problem I found was that I forgot to run the src/tools/update_po.sh and because of this I got problems with the game statistics page. Many strings in the original English pot file have changed (not so many, around 20)...
Without the update_po in the locale and locale/gui directories these ids are not updated ! This results in a mix of native language and English.

The update_po changed other languages too, but I'm submitting only pt_BR. 

Anyway, I think it worth a check in other languages too as this is almost transparent for translators.
