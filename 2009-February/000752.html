!!!HOWEVER!!!
!!WORKAROUND!!

Right click the lincity-ng.exe, select properties
Select the compatibility tab

Check "Run this program in compatibility mode for:"
and select "Windows XP (Service Pack 2)"

Then check "Run this program as an administrator"

Apply, OK.

Should work (did for me).
