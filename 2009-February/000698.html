On Sun, Feb 01, 2009 at 09:19:59PM -0800, Daniel Santos wrote:
> So anyway, I've done a little development to:
> * Add a config option for statistical display mode with the settings
> "percentage", "values" and "both" and added a new function void
> mps_store_sddp(int i, const char * s, int cur, int max) to
> src/gui_interface/mps.h and src/lincity-ng/MpsInterface.cpp to use in place
> of mps_store_sfp.  This function, based upon the config setting displays
> either a precentage (as mps_store_sfp currently does), values in the form of
> "cur/max" or both in the form of "cur/max (p%)".

When I started with lincity one year ago I did something very similar. It just didn't have the config setting. I never published a patch, because I wanted to wait till 2.0 was released. I'll attach the patch to this mail. Maybe you can use something of it. To some buildings I also added more information. E.g. Jobs and Tech-level to the light industry. The changes to stats.cpp and pbar.cpp change the display of the stats tap, so that it shows the real values. Originally the numbers were divided by some arbitrary values. The changes in stats.cpp and pbar.cpp are independent of the others and could be applied to svn separately.

> There are
> however a lot of nuances I don't quite understand yet, like the fact that the
> current get_jobs will not return true unless the 150% of the requested jobs
> are actually available.

Maybe this is also to prevent that a building which needs a lot of jobs can starve the other buildings of jobs.

> * I plan on changing the text "Jobs" for mps_market to "Workers" to properly
> reflect that this is a surplus of workers available

I'm not sure if "Workers" is the best word for it. It may be better than "Jobs" though. But when I read workers in the market, I somehow imagine that the people are taken out of their houses and stuffed in a supermarket. The concept here seems more like "saved man-days". But man-days does not sound so good. Maybe we could name it "Work" or "Labour". What do the others think ?
