Hi,

we have a preview of the upcoming 1.1.0 release of LinCity-NG 
available for download on
https://developer.berlios.de/project/showfiles.php?group_id=2929
The plan is to release mid-February. If you have anything that
should make it in the new release now would be a good time.

Especially some of the translations are outdated, you can see the
revision where the individual help files were changed on
http://svn.berlios.de/viewcvs/lincity-ng/trunk/data/help/en/?sortby=rev&sortdir=down#dirlist .
See http://svn.berlios.de/viewcvs/lincity-ng/trunk/data/help/ for
the folders with translations. If you don't compile Lincity yourself,
you can download the up to date message.pot files from 
http://prdownload.berlios.de/lincity-ng/lincity-ng-1.1.0rc2.messages_pot.tar.bz2

Because I have no idea if all translators are subscribed to 
lincity-ng-devel you might get this mail twice. Or even more often in
case you use different addresses in the .po files and in the wiki.
