I did a testimport from svn to Git like the following.

1.) git svn init -s svn://svn.berlios.de/lincity-ng
2.) git config svn.authorsfile "/some/path/to/authors" (see attachment)
3.) git svn fetch

This left me using a repository that looks quite sane (except for the fact that tags seem to end up repositories).

I'm not 100% sure on some entries in the authors file attached (maps svn usernames to real name + email). Any fixes to this are welcome.
