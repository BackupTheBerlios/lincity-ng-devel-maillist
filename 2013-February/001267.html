ATM using different images for the market would mean having two
different looking buildings which do the same thing. The modern
windmill is not just another look but also provides electricity. So
what's the idea? Use the old market for all markets while tech level
is below a certain level and then use the modern image, or let every
market keep the image from the tech era it was built in?

The later would probably make the city look more interesting but
we'd end up with different looks for the same function used at the
same time. Probably less confusing than magically upgrade/downgrade
all existing markets when techlevel changes.
