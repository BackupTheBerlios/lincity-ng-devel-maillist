Comment:
This patch fixes an issue with grid total power calculation.
If there is a power source connected at the end of the grid (as it is checked by the algorithm) and this source is connected through a substation (no direct power line path) then the grids total power is calculated equal to the last power source.
This patch fixes it.

I also changed the mill to use power when available instead of coal. Using coal by default seems kind of strange for me. Maybe there is a good rationale for it but I cant think of any.

Also added information about power line loss to the substation info panel.

I hope this patch will be useful.
