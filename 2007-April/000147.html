Hi dear maintainers and Lincity-ng people

In some files the brace placements seems very weird, and above all
very confusing to me for finding the limit of a block.
Like in commune.cpp
	}
      else
	{

So i wanted to know if there is a prefered style for coding ?
I like linux coding style (fun to read ;) because big indentation 
makes things easier to understand (and i can not argue a lot about this topic)
and brace placement.
    } else {
http://lxr.linux.no/source/Documentation/CodingStyle

There is also the tiny script Lindent which allow to automagically
reformat everything.
http://lxr.linux.no/source/scripts/Lindent
#!/bin/sh
indent -npro -kr -i8 -ts8 -sob -l80 -ss -ncs "$@"

I vote for `./Lindent src/lincity/*.cpp src/lincity/modules/*.cpp`
before next release :)
Cheers.
