Date: 2013-Jan-27 14:19
By: usr_share

Comment:
Note: as a result of using SDL_Delay (a VERY bad move), both the game's view and model were slowed down. This resulted in a severe FPS drop on all game speeds except for the fastest one. This patch uses SDL_AddTimer (SDL's callbacks) instead, resulting in a smoother view, yet keeping the gameplay slow (probably a bit faster, but still slow)
