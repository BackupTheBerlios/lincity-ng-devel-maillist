> change the variable LINCITY_LANG, but I can't find that variable.
You can use it to start lincity-ng, e.g. from a terminal, just type:
LINCITY_LANG=en lincity-ng (or how you would execute lincity-ng normally, from 
svn: LINCITY_LANG=en ./SVN/lincity-ng/lincity-ng )

Alternatively you can do those two steps:
export LINCITY_LANG=en
lincity (or how you would execute it ;) )
