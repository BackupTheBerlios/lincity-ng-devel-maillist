Summary: Some of population dies when starting new game

Details: Because water wells do not produce water immediately, some of the population dies when starting a new game unless you build the water well first and run the game until it produces water.  I think there should be a water death grace period for a year or so at the start of the game to prevent this -- that or some other mechanism.  Very nice though, thanks!

This is version 1.97 beta on Gentoo amd64 (compiled from downloaded sources).

Steps to reproduce:
1. Start a new game with an empty board
2. Pause the game (or pause before starting the new game)
3. Build a town with a water well, store and houses (as normal but in paused mode).
4. When ready, hit play (the normal speed), I loose about 10 people or due to starvation/thirst before the well kicks in and I can see it on the starvation minimap overlay.
