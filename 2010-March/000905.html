Hi,

the "MiniMap overlay mode" button cycles through five different modes.
Normal, transparent map overlay, map only, elevated, elevated with opaque
overlay. IMO a bit too much. What about an extra height on/off button?
Or is the new renderer too experimental to be part of 2.1? Then we add
the button later.

In the "New Game" screen, you either select a scenario and then
click start or you click on one of five texts below the scenario
list. Each starts the game instantly. I'd like to change that to
select and click start in all cases. Are the three "New random land"
ready for release, or are they just for testing?

The "Options" menu contains a "resolution" selector which offers
lots of resolutions. All but 1024x768 are not really well tested,
some of them are way to small to be useful, most of the are
strange. 896x672? Maybe on some laptop. While all other options are
applied at once, resolution is changed after pressing "ok". If I
select a resolution not in that list by command line that will be
the only option in the menu. Leave it as is, remove it for now? A
list with hardcoded "useful" resolution won't work given the number
of different screen formats.
