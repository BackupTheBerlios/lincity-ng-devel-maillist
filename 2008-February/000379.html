On 2008-02-26 Instruisto Jose <instruse at yahoo.com.br> wrote:
> I'm a newbie in Lincity-ng and I don't know how to configure to
> use in Brasilian Portuguese language.

Lincity automatically uses the language configured by the "locale"
setting. If you want a different setting just for Lincity you
can set the environment variable LINCITY_LANG to override the systems
language settings.

> Lincity-ng version 1.0.3

1.0.3 is too old. It does not include Brazilian Portuguese translations.

> What do you recommends, download any other package or install a
> newer version.

You can download a later version from 
http://developer.berlios.de/project/showfiles.php?group_id=2929&release_id=13664
