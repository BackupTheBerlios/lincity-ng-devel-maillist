Hi,

I think there is a bug in the pollution generation of the light industry.

In light_industry.cpp is a comment which says:
"Pollution is based on goods produced and is affected by tech level above 1000 (displayed as 100) whereupon it is reduced by one percent per 10 tech_level points."
If I understand the code correctly 1000 tech points are displayed as 0.1 or 0.1% to the user. As the code stands now the pollution from light industries is reduced starting at tech-level 0.1% till at tech-level 0.2% there is no more pollution from light industry.
I think this is the reason why I have never seen pollution from light industry. If I replace the 1000's in the code which follows the comment with MAX_TECH_LEVEL, the light industry starts to produce pollution.
See the attached patch for a possible fix.
