Merry Christmas!

we just released 1.97.beta of Lincity NG. This version has all
features of the upcoming 2.0 release. New game elements are water
management and bridges. Due to these changes most of the localized
documentation is outdated. The English version of the online help
system should be up to date which is the main reason for this beta
release.

So please check if your language is up to date. Source, Autopackage
and the Windows installer is available from
http://developer.berlios.de/project/showfiles.php?group_id=2929&release_id=15598
If you don't compile Lincity yourself you can get the messages.pot files
here: http://uafr.freeshell.org/tmp/lincity-ng-1.97.beta_locale.zip

Because not all of the translators are subscribed to lincity-ng-devel
you might get this mail twice. Or even more often in case you use different
addresses in the .po files and in the wiki.

Tschüß,
