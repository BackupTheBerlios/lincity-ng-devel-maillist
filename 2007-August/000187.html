Half a year passed since the last release of LinCity NG. The new
1.1.1 release adds some nice features to the user interface. For
better planning the range for selected buildings is shown around the
cursor. To catch the player's eye the economy tab changes colour
when people are unemployed or starving.

The online help now comes with a history function and a small
tutorial.  Localisation includes Russian and Brazilian Portuguese
online help.

Also some bugs like the problem with multi byte characters in WinXP
were fixed. Now the Sustainability progress is reloaded from saved
games and heavy industry no longer stores too much coal.

Thanks to all of you for making this new release possible.
You can download the new version from Berlios as usual.

http://lincity-ng.berlios.de/wiki/index.php/Download/Installation
https://developer.berlios.de/project/showfiles.php?group_id=2929
