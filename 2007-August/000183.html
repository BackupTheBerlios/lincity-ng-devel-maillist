Hi,

you might get this mail twice because I don't know if all
translators read the mailing list, too. LinCity NG preview is out
for two weeks, we have 1000+ downloads and no new bug report. Good
news for Windows users, the problem with multi byte characters is
fixed. So all the translated help files will work fine in Windows.
Time to prepare the next real release which will be in one or two
weeks.

New Features:
* History in online help
* Draw range for selected building around cursor 
* Online help includes a tutorial
* Russian online help
* Brazilian Portuguese online help
* Change colour of economy tab when people are unemployed or starving

Fixed Bugs
* Sustainability progress is reloaded from saved games.
* Heavy industry no longer storing too much coal.
* Fixed problem with multi byte characters in WinXP
* Reuse open statistics dialog

Is http://lincity-ng.berlios.de/wiki/index.php/Translate up to date for
your language? To spot updated help files have a look at the file age on 
http://svn.berlios.de/viewcvs/lincity-ng/trunk/data/help/
If you don't compile Lincity yourself you can get the latest .pot-files 
from http://uafr.freeshell.org/tmp/lincity_locale_1250.zip

Tschüß,
